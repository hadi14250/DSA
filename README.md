## Build & Run

Each problem directory contains:
- `README.md` - Problem description and examples
- Solution implementation file
- `main.cpp` - Test cases
- `Makefile` - Build configuration

To run a solution:
```bash
cd "Pattern/Problem Name"
make run
```

To clean build files:
```bash
make fclean
```
